# RogueCode Twitter/X Post

## English Version

I'm a solo founder who does heavy vibe coding, and I love playing games — specifically Hearthstone Battlegrounds.

The problem: Battlegrounds doesn't require much input, but I still get stuck watching the CLI waiting for AI output. And when the AI finally responds, I'm deep in the game and don't want to switch back.

So I built RogueCode.

It's a roguelike game that runs alongside your Claude Code session. Keep typing → your character fights. Go idle → the game pauses and locks you out. Every commit, every fix, every line of code feeds your run.

The goal: fill those 2-3 second gaps between AI responses with something fun, and make it impossible to get lost in a gaming rabbit hole because the game literally won't let you idle.

I just wanted a tool that respects my attention. This is it.

🎮 viberouge.ai | git clone and play
